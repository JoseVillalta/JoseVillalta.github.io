I just started reading an early access ebook copy of [The Linux Memory Manager](https://nostarch.com/linux-memory-manager) by Lorenzo Stoakes, thanks to the book bundle [Linux the Good Stuff](https://www.humblebundle.com/books/linux-good-stuff-no-starch-books). I am trying to learn all about memory management in Linux. This is deep stuff. 

One time, long ago, at a different job, I implemented malloc for an embedded system running on a custom-made RTOS. At the time I felt like a `l33t hack3r` even though I'm pretty sure I copy pasted the code from somewhere, tested that the system boots, and called it good. 
Comparing that to Linux memory management is like comparing a tree house to the Empire State Building. So anyway, I need to dive deep and really grok Linux; it comes up a lot when debugging issues at work.
